# SEO Content Writer

## Context

Use this skill when the user asks to "write SEO content", "create a blog post", "write an article", "content writing", "draft optimized content", "write me an article", "create a blog post about", "help me write SEO content", or "draft content for". Creates high-quality, SEO-optimized content that ranks in search engines.

**Trigger phrases:** "write SEO content," "create blog post," "write an article," "content writing," "draft optimized content"

---

## Instructions

### Step 1: Gather Requirements

Confirm or ask for:
- **Primary Keyword**: [main keyword]
- **Secondary Keywords**: [2-5 related keywords]
- **Target Word Count**: [length]
- **Content Type**: [blog/guide/landing page/etc.]
- **Target Audience**: [who is this for]
- **Search Intent**: [informational/commercial/transactional]
- **Tone**: [professional/casual/technical/friendly]
- **CTA Goal**: [what action should readers take]

### Step 2: Research SERP

- Analyze top results format and average word count
- Identify common sections and SERP features (snippets, PAA)
- Map keywords: primary in title/H1/intro, secondary in H2s/body

### Step 3: Create Optimized Title

- Include primary keyword (preferably at start)
- Under 60 characters for full SERP display
- Compelling and click-worthy

### Step 4: Write Meta Description

- 150-160 characters
- Include primary keyword naturally
- Include call-to-action

### Step 5: Structure Content with SEO Headers

```
# [H1 with Primary Keyword]

## [H2 - First Main Section with Secondary Keyword]
[Content...]

## [H2 - Second Main Section]
[Content...]

## Frequently Asked Questions
### [Question from PAA]?
[Answer in 40-60 words]

## Conclusion
[Summary + CTA]
```

### Step 6: Apply On-Page SEO

- Primary keyword in title, H1, first 100 words, conclusion
- Secondary keywords in H2s/H3s
- Internal links (2-5), external links (2-3 authoritative sources)
- Image alt text with keywords
- FAQ section for PAA optimization

### Step 7: Apply GEO Optimization

- Cite sources with author names and publication dates
- Include statistics with specific numbers and units
- Use quotations from experts
- Structured formatting (tables, bullet lists, clear headings)

---

## Constraints

- NEVER keyword stuff — natural integration only
- NEVER write about topics without research
- NEVER skip meta description or title optimization
- Primary keyword density should be 1-2%
- Always include FAQ section for featured snippet opportunities
